## mp-jsx

mp-jsx 是什么？它让你直接在原生小程序使用 JSX 写 WXML，实时编译。

## 立即开始

```bash
$ npm i omi-cli -g              
$ omi init-jsx my-app    
$ cd my-app        
$ npm start               
```

生成的目录和官方的模板一致，只不过多了 JSX 文件，只需要修改 JSX 文件就会实时修改 WXML。

也支持 typescript:

```bash           
$ omi init-jsx-ts my-app             
```

其他命令一样。
