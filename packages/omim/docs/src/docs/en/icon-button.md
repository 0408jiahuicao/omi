## Icon Button 

Icon button allow users to take actions, and make choices, with a single tap.

## Usage

```html
<m-icon-button icon='wifi'></m-icon-button>
```


## API

### Props

```jsx
{
  ripple?: boolean,
  toggle?: object,
  icons: object,
  icon: string,
  imgs: object,
  svg: object,
  color: string,
  disabled: boolean
}
```
