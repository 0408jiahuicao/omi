## Badge 

The number of circular logo in the upper right corner of the icon.

## 使用

```js
<o-badge number={8}>
  <o-icon type="chat" isFill color="#07C160" />
</o-badge>
```

## API

### Props

|  **Name**  | **Type**        | **Defaults**  | **Details**  |
| ------------- |:-------------:|:-----:|:-------------|
| number         |   number  |    --   |             |　
| children         |    json array  |    --   |   The array's length is 1.        |
